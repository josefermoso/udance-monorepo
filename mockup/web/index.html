🔧 Consideraciones Técnicas - Web App

  • Layout: Desktop-first con navegación horizontal, sidebar filters
  • Performance: Server-side rendering para SEO, image optimization
  • SEO: Meta tags dinámicos, structured data para eventos
  • Accesibilidad: ARIA labels, keyboard navigation, screen reader support
  • Shared Components: Header, Footer, EventCard, PlaceCard, ArticleCard
  • Unique Components: Calendar View, Map Integration, Advanced Filters
  • Data Loading: SSR para landing, CSR para interacciones, ISR para contenido
  • Authentication: Reutiliza el sistema OAuth ya configurado
