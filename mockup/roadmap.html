Detailed MVP Breakdown

MVP 1: Core Platform & Community Building

Focus: Establish the foundational community. Get users, instructors, and studios on the platform and interacting through core discovery and profile features.

Student

Create a profile, search for events/classes, view instructor/studio profiles, and express interest in events.

Instructor

Create a detailed professional profile, list classes with schedules, and link to studios.

Studio Admin

Create and manage a detailed studio profile, listing amenities, photos, and classes.

Event Organizer

Create and publish a detailed event page and view a list of interested users.

Platform Admin

Manage user roles and perform basic user lookups.

MVP 2: Monetization & Admin Power Tools

Focus: Introduce revenue streams for organizers and the platform, and provide robust administrative tools for control and moderation.

Student

Purchase tickets with QR codes and leave detailed reviews for events, classes, and instructors.

Instructor

View public reviews, receive booking notifications, and access a basic analytics dashboard.

Studio Admin

Manage financial agreements with the platform and view manual payment records.

Event Organizer

Enable ticketing with QR code scanning, track ticket sales, and manage event finances.

Platform Admin

Full dashboard to manage taxonomy, moderate content, view audit logs, and manage finances.

Future Enhancements

Focus: Long-term growth, user engagement, and platform intelligence.

Advanced Analytics

Comprehensive dashboards for all power-user roles.

Automated Notifications

Robust system for email and push notifications.

Subscription Models

Monthly/annual plans for students and studios.

Direct Messaging

Allow users to connect and communicate directly.

Gift Cards & Promotions

System for creating and managing promotions.

Search & Recommendations

Personalized recommendation engine for classes and events.
