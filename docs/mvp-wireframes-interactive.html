🧩 Componentes Base del Sistema

💡 Interacción: Haz clic en cada componente para ver su wireframe detallado.
1

🔍 Componentes de Búsqueda

💬 ChatInput IA
Input inteligente: "Quiero aprender algo caribeño y divertido" → Sugiere Salsa/Bachata
🔍 ResultsGrid
Grid de clases con filtros: estilo, zona, horario, nivel, instructor
🗺️ MapComponent
Mapa interactivo con marcadores de estudios, filtros integrados, vista satelital
⚙️ FilterSidebar
Panel de filtros: estilos de baile, fechas, ubicación, nivel, precio
2

📋 Componentes de Reserva

📝 BookingForm
Formulario simple: nombre, teléfono, nivel de baile. SIN pagos online.
📚 ClassCard
Card de clase: horario, instructor, nivel, precio, botón "Reservar"
👨‍🏫 InstructorCard
Perfil de instructor: foto, especialidades, rating, horarios disponibles
🏢 StudioCard
Card de estudio: fotos, ubicación, clases disponibles, contacto directo
3

🎪 Componentes de Eventos

🎉 EventCard
Card de evento con imagen de fondo, fecha visible, información del instructor, ubicación, precios. Gradiente border arcoíris en hover, efecto zoom en imagen, overlay oscuro interactivo. Tamaños: small (256px), medium (512px), large (768px). Layout vertical y horizontal disponibles.
📱 QR Generator
Generación de QR para eventos, visualización, descarga, check-in
📅 EventList
Lista de eventos: próximos, filtros por estilo, vista calendario
✅ AttendanceTracker
Seguimiento de asistencia via QR, listas en tiempo real
4

🎨 Componentes UI Base

🧭 NavBar
Navegación principal: logo, búsqueda, usuario, menú responsive
📰 NewsCard
Card de noticia con imagen de fondo, título superpuesto, gradiente rainbow border en hover, efecto zoom interactivo. Tamaños: main-article (640px), sub-article (350px), other-big (350x180px), other-article (200x180px). Layouts vertical y horizontal disponibles.
⭐ ReviewComponent
Sistema de reseñas: estrellas, comentarios, verificación
🔔 NotificationCenter
Centro de notificaciones: alertas, confirmaciones, recordatorios

Tamaños disponibles:

  • main-article: 640x320px - Artículo principal destacado
  • sub-article: 350x320px - Artículo secundario importante
  • other-big: 350x180px - Otros artículos grandes
  • other-article: 200x180px - Artículos pequeños

Uso en wireframes:

  • Homepage: Grid 2fr 1fr con main/sub articles
  • Grid lateral: 4 artículos other-article en 2x2
  • Carrusel: NewsCards horizontales
  • Detalle relacionados: other-big articles

Integración con database:

Se conecta a la tabla news_articles para obtener:
• Imagen del artículo (article_image_url)
• Título y resumen
• Fecha de publicación (published_date)
• Autor (author_name)
• Categoría (category)
• Tags relacionados
